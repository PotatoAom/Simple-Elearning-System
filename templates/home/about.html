E-Learning Management System

เว็ปแอปพลิเคชั่น E-Learning Management System จัดทำขึ้นโดยมีวัตถุประสงค์ที่จะศึกษาวิธีการสร้าง Web Application และโครงสร้างต่างๆของ Django Framework ซึ่งเขียนด้วยภาษา Python และฝึกการออกแบบระบบ Web Application เพื่อนำไปต่อยอดในการสร้างและออกแบบระบบ Web Application สามารถนำไปประยุกต์ใช้ในการพัฒนาเว็บแอปพลิเคชั่นในอนาคต

โปรเจคนี้มีวัตถุประสงค์ในการศึกษาการสร้างเว็ปแอพ (This project made for education purpose)

สามารถนำไปต่อยอดหรือนำไปพัฒนาต่อได้เลย แนบลิ้งไว้ที่ footer (logo Github)

Project Objectives

  • เพื่อศึกษาวิธีการใช้งาน Django Framework
  • เพื่อฝึกการเขียนโปรแกรมด้วยภาษา Python
  • ฝึกการออกแบบระบบ Web Applications
  • เพิ่มช่องทางในการศึกษารูปแบบใหม่
  • สามารถนำไปต่อยอดการพัฒนาเว็บแอปพลิเคชั่นต่อไปในอนาคต
